# Doland-r-c-lar-n-G-ncel-Sayfalar-
Bu platform, T.C. vatandaşlarının dijital güvenliğini sağlamak ve siber dolandırıcılık faaliyetlerine karşı toplumsal farkındalık oluşturmak amacıyla kurulmuştur. Aşağıdaki liste, somut deliller ve kapsamlı araştırmalar neticesinde "şüpheli/dolandırıcı" olarak sınıflandırılan mecraları içermektedir.
